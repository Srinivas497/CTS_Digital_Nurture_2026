
# Week 6 – Git Version Control

## Overview

This directory contains the hands-on exercises completed as part of the **Cognizant Deep Skilling Program – Week 6 (Git Module)**.

The exercises focus on learning Git fundamentals, repository management, branching strategies, merge conflict resolution, and synchronization with remote repositories using GitHub.

---

## Technologies Used

- Git
- GitHub
- Git Bash
- Visual Studio Code

---

## Hands-on Exercises

### 1. Git HOL – Git Basics

- Git configuration
- Repository initialization
- Git status
- Staging files
- Commit history
- Push and Pull operations

---

### 2. Git HOL – Using .gitignore

- Ignoring unnecessary files
- Ignoring log files
- Ignoring directories
- Verifying ignored files

---

### 3. Git HOL – Branching and Merging

- Branch creation
- Switching branches
- Branch comparison
- Merging branches
- Viewing commit history
- Branch deletion

---

### 4. Git HOL – Merge Conflict Resolution

- Creating merge conflicts
- Resolving conflicts
- Updating `.gitignore`
- Completing merges
- Repository cleanup

---

### 5. Git HOL – Cleanup and Remote Synchronization

- Repository status verification
- Pull latest changes
- Push local commits
- Verify repository synchronization
