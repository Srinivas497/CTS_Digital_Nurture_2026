# Git HOL 1 – Git Basics

## Objective

Learn the fundamental Git commands used for version control and repository management.

## Topics Covered

- Git Installation Verification
- Git Configuration
- Repository Status
- Staging Files
- Creating Commits
- Viewing Commit History
- Push and Pull Operations

## Commands Practiced

```bash
git --version
git config --global user.name
git config --global user.email
git status
git add .
git commit
git log --oneline
git push
git pull
```
