# Git HOL 3 – Branching and Merging

## Objective

Understand Git branching and merging workflows.

## Topics Covered

- Create Branch
- Switch Branch
- Commit Changes
- Compare Branches
- Merge Branches
- Delete Branch
- View Commit Graph

## Commands Practiced

```bash
git branch
git checkout
git add
git commit
git diff
git merge
git log --oneline --graph --decorate
git branch -d
```

## Files

- GitPractice/
  - branch.txt
  - README.md

## Learning Outcome

Successfully created a branch, committed changes, merged them into the main branch, and deleted the merged branch.
