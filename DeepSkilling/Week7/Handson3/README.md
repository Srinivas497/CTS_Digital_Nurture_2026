# Handson 3 – React Score Calculator

## Objective

Implement a React application to calculate and display a student's score using reusable components, JSX, props and CSS styling.

## Steps Performed

- Created React application `scorecalculatorapp`
- Created `CalculateScore` component.
- Used JSX to display student details.
- Passed data using props.
- Calculated percentage dynamically.
- Applied CSS styling using an external stylesheet.
- Rendered component in `App.jsx`.
- Executed the application using `npm run dev`.

## Output

The application displays:

- Student Name
- School Name
- Total Marks
- Calculated Percentage
