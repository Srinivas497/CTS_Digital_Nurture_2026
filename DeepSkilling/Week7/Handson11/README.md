# Handson 11 – React Event Handling

## Objective
Implement event handling in React using buttons, event handlers, synthetic events, and a simple currency converter.

## Features
- Increment Counter
- Decrement Counter
- Invoke multiple functions using one button
- Display "Hello! Member1"
- Display Welcome message
- Handle Synthetic Events
- Currency Converter component
- Form submission using React event handling

## Technologies Used
- React
- JSX
- JavaScript
- CSS
- Vite

## Steps Performed
- Created Increment and Decrement buttons.
- Implemented multiple event handlers.
- Displayed welcome messages using event arguments.
- Implemented synthetic click event.
- Created Currency Converter component.
- Handled form submission using onSubmit.

## Output
The application successfully demonstrates React event handling, synthetic events, and currency conversion.
