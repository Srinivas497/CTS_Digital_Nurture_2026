# Handson 9 – ES6 Features in React

## Objective
Develop a React application demonstrating ES6 features such as map(), arrow functions, destructuring, spread operator, and conditional rendering.

## Concepts Covered

- ES6 map()
- Arrow Functions
- Array Filtering
- Destructuring
- Spread Operator
- Conditional Rendering
- Functional Components
- JSX

## Steps Performed

- Created React application using Vite.
- Displayed player details using ES6 map().
- Filtered players with scores below 70.
- Used arrow functions.
- Implemented array destructuring.
- Merged arrays using the spread operator.
- Used conditional rendering with a flag variable.

## Output

Successfully displayed:

- List of Players
- Players scoring below 70
- Odd Team Players
- Even Team Players
- Merged Indian Players
