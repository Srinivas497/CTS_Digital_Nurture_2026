# Handson 5 – Cohort Tracker

## Objective
Develop a React application to display Cohort details using reusable components, CSS Modules, props, and conditional styling.

## Concepts Covered
- React Functional Components
- Props
- CSS Modules
- Component Reusability
- Conditional Styling
- JSX

## Steps Performed

- Created React application using Vite.
- Created reusable CohortDetails component.
- Passed cohort details through props.
- Applied CSS Modules for styling.
- Displayed multiple cohort cards.
- Used conditional colors for Scheduled and Ongoing batches.

## Output

Successfully displayed all cohort details in styled cards.
