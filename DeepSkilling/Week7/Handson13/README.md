# Handson 13 – React Conditional Rendering

## Objective

Implement conditional rendering in React by displaying multiple components and rendering lists using the `map()` function with unique keys.

---

## Project Name

bloggerapp

---

## Components Created

- BlogDetails
- BookDetails
- CourseDetails

---

## Concepts Covered

- Functional Components
- Conditional Rendering
- JSX
- List Rendering
- React map() function
- Keys in React
- Component Composition

---

## Steps Performed

- Created a new React application named **bloggerapp**.
- Created separate components for Blogs, Books, and Courses.
- Stored sample data in a shared `data.js` file.
- Rendered each dataset using the React `map()` function.
- Assigned unique keys to each rendered element.
- Displayed all three components side-by-side using Flexbox.
- Applied CSS styling for layout and presentation.
- Executed the application using `npm run dev`.

---

## Output

The application displays:

- Course Details
- Book Details
- Blog Details

Each section renders data dynamically using React list rendering and conditional rendering techniques.
