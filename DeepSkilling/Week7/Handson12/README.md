# Handson 12 – Conditional Rendering (Ticket Booking App)

## Objective
Implement Conditional Rendering in React by displaying different pages for Guest and Logged-in users.

## Features
- Guest Page
- User Page
- Login Button
- Logout Button
- Conditional Rendering using state
- React Functional Components

## Technologies Used
- React
- JSX
- JavaScript
- CSS
- Vite

## Steps Performed
- Created GuestGreeting component.
- Created UserGreeting component.
- Created Login button.
- Created Logout button.
- Used React useState() for login status.
- Implemented conditional rendering using if statement and ternary operator.

## Output
The application displays the Guest page initially. Clicking Login displays the User page. Clicking Logout returns to the Guest page.
