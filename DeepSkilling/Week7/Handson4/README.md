# Handson 4 – Blog Application

## Objective
Develop a React Blog Application using reusable React components.

## Technologies Used
- React
- JavaScript (JSX)
- CSS
- Vite
- Visual Studio Code

## Folder Structure

Handson4/
├── blogapp/
├── Screenshots/
└── README.md

## Steps Performed

- Created a React application.
- Developed reusable Post component.
- Created Posts component to render multiple blog posts.
- Imported Posts component into App.jsx.
- Styled the application using CSS.
- Executed the application using npm run dev.

## Output

The application displays multiple blog posts using reusable React components.
