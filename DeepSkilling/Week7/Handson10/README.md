# Handson 10 – Office Space Rental App

## Objective
Develop a React application that displays office space information using JSX, objects, arrays, conditional rendering, and styling.

## Features
- Display office space details.
- Display office image.
- Display office name.
- Display office rent.
- Display office address.
- Display multiple office spaces using `map()`.
- Display rent in:
  - Red (Rent ≤ 60000)
  - Green (Rent > 60000)

## Technologies Used
- React
- JSX
- CSS
- Vite

## Steps Performed
- Created Office objects.
- Stored office objects inside an array.
- Rendered the list using `map()`.
- Applied conditional styling for rent.
- Displayed office images using React assets.

## Output
The application successfully displays all office details with conditional rent coloring.
